**`PATCH /api/orders/:id/status`** — Updates an order's status.

- **Request body**: `{ "status": "confirmed" }` (or `cancelled`, `shipped`, `delivered`, `pending`)
- **404** if the order doesn't exist or is soft-deleted
- **400** if the status is missing or not one of the valid values
- Returns the updated order on success

The valid statuses are the ones already defined in `validators.js`: `pending`, `confirmed`, `shipped`, `delivered`, `cancelled`.
