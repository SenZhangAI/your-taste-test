Here's a summary of all changes:

**`src/services/order-service.js`** — `listOrders` queries use `.whereNull('deleted_at')` instead of `.where('status', '!=', 'deleted')`; `softDeleteOrder` sets `deleted_at` instead of `status: 'deleted'`

**`src/routes/orders.js`** — GET `/:id` and DELETE `/:id` check `order.deleted_at` instead of `order.status === 'deleted'`

**`src/routes/users.js`** — List users, get-user, and user's orders queries all migrated to `.whereNull('deleted_at')` / `user.deleted_at`

**`src/routes/products.js`** — List products and get-product migrated to `.whereNull('deleted_at')` / `product.deleted_at`

**`src/seed.js`** — Soft-deleted seed orders now set `deleted_at` timestamp instead of `status: 'deleted'`, keeping a valid status for all rows
