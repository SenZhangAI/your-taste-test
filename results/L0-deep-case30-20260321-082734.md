**`POST /api/orders/bulk`** — creates multiple orders in a single request.

**Request body:**
```json
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "total_cents": 999, "quantity": 2 },
    { "user_id": 2, "product_name": "Gadget", "total_cents": 1499 }
  ]
}
```

**Behavior:**
- Validates that `orders` is a non-empty array
- Validates each order has required fields (`user_id`, `product_name`, `total_cents`), returning a 400 with the failing index if not
- Inserts all orders in a **database transaction** — if any insert fails, they all roll back
- Returns `201` with `{ orders: [...] }` containing the created order records
