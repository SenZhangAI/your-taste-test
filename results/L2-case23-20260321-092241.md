**`POST /api/orders/import`** — accepts a CSV file upload (form field `file`).

**Route** (`src/routes/orders.js:34`):
- Multer handles multipart upload (memory storage, 5MB limit)
- Auto-detects and skips header row
- Validates each row: all 5 columns present, quantity/total_cents are positive integers, date is parseable
- Resolves `user_email` → `user_id` via DB lookup (batch query, not N+1)
- Returns per-row errors for invalid/unresolvable rows without failing the whole import

**Service** (`src/services/order-service.js`):
- `bulkCreateOrders()` inserts all valid rows in a single transaction
- Sets `created_at` from the CSV date, status defaults to `pending`

**Response format** (201):
```json
{
  "imported": 42,
  "failed": 3,
  "errors": [
    { "row": 5, "error": "User not found: bad@email.com" }
  ]
}
```

**Usage:**
```bash
curl -X POST http://localhost:3000/api/orders/import \
```
