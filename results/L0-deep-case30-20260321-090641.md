**`POST /api/orders/bulk`** — accepts `{ orders: [...] }` in the request body.

**Route** (`src/routes/orders.js:53-73`):
- Validates the body contains a non-empty `orders` array
- Validates each order using the existing `validateOrderPayload` (requires `user_id`, `product_name`, `total_cents`)
- Returns per-item validation errors with the array index for easy debugging
- Returns `201` with all created orders on success

**Service** (`src/services/order-service.js:71-87`):
- `createBulkOrders()` wraps all inserts in a Knex transaction — if any insert fails, the entire batch rolls back
- Returns all created order records
