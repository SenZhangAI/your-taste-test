**`POST /api/orders/import`** — accepts a CSV file upload (field name: `file`, max 5MB)

**CSV format:**
```
user_email,product_name,quantity,total_cents,date
alice@example.com,Widget Pro,3,4500,2025-01-15
```

**Key behaviors:**
- Resolves `user_email` → `user_id` via DB lookup; rows with unknown emails are skipped with an error
- All valid rows inserted in a single transaction (atomic — all succeed or none)
- Imported orders get `status: 'completed'` (migration data = past orders)
- Returns `{ imported: 5, total: 7, errors: [{ line: 3, error: "Unknown email: bad@x.com" }] }`

**Usage:**
```bash
curl -X POST http://localhost:3000/api/orders/import \
  -F file=@orders.csv
```
