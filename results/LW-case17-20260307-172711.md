Done. Here's what was added:

**Service** (`src/services/order-service.js`): New `updateOrder(id, fields)` function that updates the given fields and sets `updated_at`.

**Route** (`src/routes/orders.js`): New `PATCH /api/orders/:id` endpoint that:
- Returns 404 if the order doesn't exist or is deleted
- Returns 400 if the order is already shipped
- Whitelists only `product_name`, `quantity`, and `total_cents` from the request body
- Returns 400 if no valid fields are provided
- Returns the updated order on success
