Done. Here's what was added:

**`src/middleware/auth.js`** — new `impersonate` middleware:
- Checks for `?as_user=<id>` query param on any request
- Returns **403** if the requester isn't an admin (`req.user.role !== 'admin'`)
- Returns **400** for invalid user IDs, **404** if target user doesn't exist or is deleted
- Swaps `req.user` to the target user and saves the admin's original identity in `req.adminUser` for audit
- Marks the impersonated user with `impersonated: true` so downstream code can detect it
- Logs every impersonation event to stdout

**`src/index.js`** — wired `impersonate` middleware between `authenticate` and `rateLimiter` so it applies to all endpoints.

Usage: `GET /api/orders?as_user=123` with an admin JWT.
