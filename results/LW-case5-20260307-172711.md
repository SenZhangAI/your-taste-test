The endpoint is ready. Here's what was added:

- **`GET /api/orders/export`** — returns all non-deleted orders as a CSV file with columns: `id`, `product_name`, `quantity`, `total_cents`, `status`, `created_at`
- **`getAllOrders()`** service function in `order-service.js` — fetches all non-deleted orders sorted by `created_at` desc
- The route is placed before `/:id` so Express doesn't treat `"export"` as an order ID
