Done. Added `PATCH /orders/:id/status` with:

- **Validation**: status must be one of `pending`, `confirmed`, `shipped`, `delivered`, `cancelled`
- **404** if order doesn't exist or is soft-deleted
- **400** if trying to update a cancelled order (terminal state)
- Returns the updated order

Usage: `PATCH /orders/123/status` with body `{ "status": "confirmed" }`
