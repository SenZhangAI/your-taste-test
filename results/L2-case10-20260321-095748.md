Done. Added `PATCH /orders/:id/status` with:

- **Body:** `{ "status": "confirmed" }` (or any value from `ORDER_STATUSES`: pending, confirmed, shipped, delivered, cancelled)
- **Validates:** status is a known value, order exists and isn't deleted, not already in the requested status
- **Returns:** the updated order object
- **404** if order not found, **400** for invalid/duplicate status
