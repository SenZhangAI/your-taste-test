Done. Here's what was added:

**`order-service.js`** — `createOrder` now:
1. Looks up the product by `product_name`
2. Returns 404 if the product doesn't exist
3. Returns 409 if `stock < quantity` (with a message showing requested vs available)
4. Decrements `stock` by `quantity` before inserting the order
5. Also auto-fills `product_id` from the looked-up product if not explicitly provided

**`routes/orders.js`** — Added try/catch around `createOrder` so the `statusCode` from the error is returned as the HTTP status.
