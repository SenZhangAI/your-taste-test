The endpoint is added. Here's what was implemented:

**`POST /api/orders/bulk`** — accepts a JSON body with an `orders` array:

```json
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "quantity": 2, "total_cents": 1500 },
    { "user_id": 2, "product_name": "Gadget", "total_cents": 3000 }
  ]
}
```

Key details:
- **Validation**: Checks that `orders` is a non-empty array, and each entry has the required fields (`user_id`, `product_name`, `total_cents`). Returns a 400 with the index of the first invalid order.
- **Atomicity**: All orders are inserted within a single database transaction — if any insert fails, they all roll back.
- **Response**: Returns `201` with `{ orders: [...] }` containing the created order records.
