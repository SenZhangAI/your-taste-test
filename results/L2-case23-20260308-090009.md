Done. Here's what was added:

**`POST /api/orders/import`** — accepts a CSV file via multipart form (`field: "file"`).

**Route** (`src/routes/orders.js`):
- Uses `multer` (memory storage, 5MB limit) to receive the upload
- Validates file presence, header row with required columns
- Parses CSV respecting header column order (not positional)
- Returns `201` on full success, `207` on partial success with per-row errors

**Service** (`src/services/order-service.js` → `importOrders`):
- Batch-resolves `user_email` → `user_id` in one query
- Validates each row: required fields, valid `total_cents`, user exists
- Inserts with `status: 'completed'` and preserves the CSV `date` as `created_at`
- Returns `{ imported: number, errors: [{ line, error }] }`

**Usage:**
```bash
curl -X POST http://localhost:3000/api/orders/import \
  -F "file=@orders.csv"
```

CSV format:
```
user_email,product_name,quantity,total_cents,date
alice@example.com,Widget,2,5000,2024-01-15
bob@example.com,Gadget,1,12000,2024-02-20
```
