**`POST /api/orders/bulk`** — accepts `{ orders: [...] }` in the request body.

**Request format:**
```json
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "quantity": 2, "total_cents": 1500 },
    { "user_id": 1, "product_name": "Gadget", "total_cents": 3000 }
  ]
}
```

**Behavior:**
- Validates all orders upfront — returns 400 with per-item error details (`index` + message) if any fail
- Inserts all orders in a **single transaction** — if any insert fails, the entire batch rolls back
- Returns `201` with `{ orders: [...] }` containing the created order objects
