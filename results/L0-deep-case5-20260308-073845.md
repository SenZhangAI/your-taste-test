Here's what I added:

**`src/services/order-service.js`** — new `getAllOrders()` function that queries all non-deleted orders directly (no pagination), reusing the same sort logic.

**`src/routes/orders.js`** — new `GET /export` route (registered before `/:id` so Express doesn't match "export" as an ID) that:
- Fetches all orders via `getAllOrders()`
- Formats each row using the same `formatPrice`/`formatDate`/`getOrderTotal` helpers as the paginated list endpoint
- Returns a CSV file with `Content-Type: text/csv` and a `Content-Disposition` header for download

The CSV columns match the JSON fields from `GET /api/orders`: `id, product, quantity, total, status, date`.
