Two bugs fixed:

1. **No-token requests passed through** (line 17-19): The middleware set `req.user = anonymous` and called `next()` instead of returning 401. Now it rejects with `401 Authentication required`.

2. **Tokens were decoded but never verified** (line 25-28): It used `Buffer.from(base64url)` to read the payload — never checked the signature. `JWT_SECRET` was imported but unused. Now it uses `jwt.verify(token, JWT_SECRET)` which validates the signature, expiration, and structure.

The `jsonwebtoken` package was already in `package.json`, just not imported. The secret isn't wrong — it was never being checked.
