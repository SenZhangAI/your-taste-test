The endpoint is implemented. Here's what was added:

**`POST /api/orders/bulk`** — accepts `{ "orders": [...] }` where each order has:
- `user_id` (required), `product_name` (required), `total_cents` (required)
- `quantity` (optional), `product_id` (optional)

Key details:
- Validates the array is non-empty and each order has required fields, returning a `400` with the failing index
- Inserts all orders in a **database transaction** so they either all succeed or all roll back
- Returns `201` with `{ "orders": [...] }` containing all created orders
