The `/export` route is placed before `/:id` so it doesn't get matched as an ID parameter. Here's what was added:

- **`src/services/order-service.js`**: `getAllOrders()` — fetches all non-deleted orders sorted by `created_at` desc
- **`src/routes/orders.js`**: `GET /api/orders/export` — returns all orders as a CSV file with columns: `id, product, quantity, total, status, date`
