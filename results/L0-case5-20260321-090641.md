**`src/services/order-service.js`** — New `getAllOrders({ sort, status })` function that queries all non-deleted orders without pagination.

**`src/routes/orders.js`** — New `GET /api/orders/export` endpoint that:
- Queries all orders via `getAllOrders` (no page looping needed — single DB query)
- Builds CSV with columns: `id, product, quantity, total, status, date`
- Properly escapes product names with double-quote CSV escaping
- Sets `Content-Type: text/csv` and `Content-Disposition: attachment` headers
- Supports optional `?status=` and `?sort=` query params (same as the list endpoint)
- Is registered **before** the `/:id` route so Express doesn't match "export" as an ID
