**`POST /api/orders/bulk`** — accepts `{ "orders": [...] }` where each order has `user_id`, `product_name`, `total_cents` (required) plus optional `quantity`, `product_id`.

- Validates the array is non-empty and each entry has required fields
- Inserts all orders in a single DB transaction (all-or-nothing)
- Returns `201` with `{ "orders": [...] }` containing the created records

Example request body:
```json
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "total_cents": 999, "quantity": 2 },
    { "user_id": 2, "product_name": "Gadget", "total_cents": 1499 }
  ]
}
```
