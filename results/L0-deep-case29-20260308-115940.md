Done. Here's what was added to `src/middleware/auth.js`:

- **`?as_user=123`** query param on any endpoint triggers impersonation
- **Admin-only**: returns `403` if the authenticated user's role isn't `admin`
- **Validates target user**: returns `404` if the user ID doesn't exist in the database
- **Swaps `req.user`** to the target user so all downstream route handlers act as that user
- **Preserves audit trail**: the original admin identity is saved on `req.impersonatedBy`, and `req.user.impersonated` is set to `true` — useful for logging/auditing

Since the middleware runs before all route handlers, this works automatically on every endpoint without any per-route changes.
