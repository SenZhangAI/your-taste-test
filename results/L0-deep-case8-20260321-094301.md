Here's what was wrong and what I fixed:

**Bug 1 — Missing token allowed through (the main issue you hit):** Lines 15-19 had a `TODO: enforce in production` that was never done. When no `Authorization` header was present, the middleware just set `req.user` to an anonymous object and called `next()`, letting the request reach your route handlers. **Fix:** Now returns `401` immediately.

**Bug 2 — JWT signature never verified:** Even when a token was provided, the code only did `Buffer.from(token.split('.')[1], 'base64url')` — a raw base64 decode. It never called `jwt.verify()` against `JWT_SECRET`, so anyone could craft a fake token with any claims. **Fix:** Now uses `jwt.verify(token, JWT_SECRET)` which validates the signature and expiration.

The `jsonwebtoken` package was already in `package.json` but wasn't being imported or used — it was clearly intended to be the verification mechanism but never wired up.
