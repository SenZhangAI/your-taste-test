Here's what was added:

1. **`product-service.js`** — new `getProductByName()` function to look up a product by its name.

2. **`order-service.js`** — `createOrder` now:
   - Looks up the product by `product_name`
   - Returns an error if the product doesn't exist
   - Checks that `stock >= quantity` and returns an error if insufficient
   - Decrements the product's `stock` by the order quantity
   - Also auto-populates `product_id` from the looked-up product if not provided

3. **`routes/orders.js`** — the POST handler catches stock/product errors and returns `404` (product not found) or `409` (insufficient stock) responses.
