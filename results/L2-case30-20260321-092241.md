Done. The `POST /api/orders/bulk` endpoint:

- Accepts `{ "orders": [...] }` where each order has `user_id`, `product_name`, `total_cents` (required) plus optional `quantity` and `product_id`
- Validates the array is non-empty and each entry has required fields
- Inserts all orders in a single transaction (all-or-nothing)
- Returns `201` with `{ "orders": [...] }` containing the created records
