The endpoint is working. Here's what was added:

**`POST /api/orders/bulk`** — accepts a JSON body with an `orders` array:

```json
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "total_cents": 999, "quantity": 2 },
    { "user_id": 2, "product_name": "Gadget", "total_cents": 1499 }
  ]
}
```

Key details:
- **Validation**: Checks that `orders` is a non-empty array and each item has `user_id`, `product_name`, and `total_cents`. Returns all validation errors at once with their array index.
- **Atomic**: Uses a database transaction so either all orders are created or none are.
- **Response**: Returns `201` with `{ orders: [...] }` containing all created order objects.
- Each order defaults to `quantity: 1` and `status: 'pending'`, matching the single-create behavior.
