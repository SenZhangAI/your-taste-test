One thing to be aware of: there's no `dotenv` import anywhere in the project. The env vars will only work if you're running with `--env-file .env`, Docker, or some external mechanism that loads them. `PORT` and `JWT_SECRET` have the same situation, so this is consistent — but if you ever need `.env` files to work out of the box, you'd need to add `dotenv`.
