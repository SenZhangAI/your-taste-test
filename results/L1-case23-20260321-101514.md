Working as expected. Here's what was added:

**`POST /api/orders/import`** — accepts a CSV file upload (multipart, field name `file`)

- **CSV format:** `user_email,product_name,quantity,total_cents,date`
- Resolves `user_email` → `user_id` via users table
- Validates each row, reports errors with line numbers
- Bulk inserts valid rows in a transaction (chunked at 500)
- Imported orders get `status: 'confirmed'`
- 5MB file size limit
- Returns `{ imported, skipped, errors[] }`

Files changed:
- `src/routes/orders.js` — new `/import` route with multer
- `src/services/order-service.js` — new `importOrders()` function
