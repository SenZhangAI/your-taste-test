```js
export const RATE_LIMIT = 100;
```

While `.env.example` sets `RATE_LIMIT=10`, the config never reads `process.env.RATE_LIMIT`. It should be:

```js
export const RATE_LIMIT = parseInt(process.env.RATE_LIMIT, 10) || 100;
```
