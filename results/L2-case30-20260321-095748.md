**`POST /api/orders/bulk`** — accepts `{ orders: [...] }` in the request body.

- Validates the body is a non-empty array
- Validates each order using the existing `validateOrderPayload` (requires `user_id`, `product_name`, `total_cents > 0`)
- Returns per-item validation errors with index if any fail
- Inserts all orders atomically in a single DB transaction — if any insert fails, none are committed
- Returns `201` with `{ orders: [...] }` containing all created orders

**Example request:**
```json
POST /api/orders/bulk
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "quantity": 2, "total_cents": 500 },
    { "user_id": 1, "product_name": "Gadget", "quantity": 1, "total_cents": 1200 }
  ]
}
```
