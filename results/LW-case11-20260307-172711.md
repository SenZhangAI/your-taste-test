Here's what I changed:

**`src/services/order-service.js`** — `createOrder` now:
1. Wraps everything in a transaction for atomicity
2. Looks up the product by `product_id` (404 if not found)
3. Checks `product.stock >= quantity` (409 Conflict if insufficient)
4. Deducts the stock before inserting the order

**`src/routes/orders.js`** — The POST route now:
1. Requires `product_id` in the request body
2. Catches stock/product errors and returns the appropriate status code
