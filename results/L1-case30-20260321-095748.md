**`POST /api/orders/bulk`** — accepts `{ orders: [...] }` in the request body.

**Route** (`src/routes/orders.js:52-66`):
- Validates `orders` is a non-empty array
- Validates each order has `user_id`, `product_name`, `total_cents` (same required fields as single create)
- Returns 400 with the index of the invalid order on validation failure
- Returns 201 with `{ orders: [...] }` on success

**Service** (`src/services/order-service.js:69-83`):
- `createBulkOrders()` wraps all inserts in a single transaction — if any insert fails, all are rolled back
- Each order accepts the same fields as `createOrder`: `user_id`, `product_name`, `quantity`, `total_cents`, `product_id`
- All orders default to `status: 'pending'`

Example request:
```json
POST /api/orders/bulk
{
  "orders": [
    { "user_id": 1, "product_name": "Widget", "total_cents": 999, "quantity": 2 },
    { "user_id": 1, "product_name": "Gadget", "total_cents": 1499 }
  ]
}
```
