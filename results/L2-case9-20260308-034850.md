Here's a summary of what was leaking and the fixes:

| File | Problem | Fix |
|------|---------|-----|
| `middleware/rate-limiter.js` | `requestLog` array — every request appended, never cleaned | Replaced with a simple `requestCount` counter. Removed `last100` from stats. |
| `middleware/logger.js` | `recentErrors` array — every error appended, never trimmed | Cap at 50 entries, `shift()` oldest when full. |
| `services/user-service.js` | `userCache` Map — entries added forever | Evict oldest entry when cache exceeds 1000. |
| `services/product-service.js` | `priceCache` object — entries added forever | Evict oldest entry when cache exceeds 500. |

The **`requestLog`** was by far the worst — it stored a full object `{ip, timestamp, path}` for every single request with zero cleanup. On a busy server this would consume hundreds of MB within days.
